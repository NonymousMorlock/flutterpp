# FlutterPP [Quick Video Demo](https://youtu.be/j8yr95t9LIM)

Welcome to FlutterPP, your developer companion for building Flutter apps with ease.

## Overview

FlutterPP (Flutter Plus Plus) is a low-code tool designed to significantly reduce development time by automating code generation for Flutter applications. This initial release, marks the beginning of an exciting journey towards making Flutter development more efficient and accessible.

# Flutter Plus Plus Features
Flutter Plus Plus can automatically generate project files using BLoC OR GetX frameworks ( working on adding more! ). It creates data classes (models), routes (GoRouter & GetX Router), controllers, services, providers, views, and pages, all following clean code principles inspired by Uncle Bob and MVC architecture.

- **Data Models:** Easily generate data classes that represent your application's data structures, ensuring consistency and simplicity in your codebase.
- **Routes:** Automatically create routing configurations using GoRouter or GetX Router, making navigation within your app seamless and organized.
- **Controllers:** Generate controllers to manage the logic and state of your application, providing a clean separation of concerns.
- **Services:** Set up service layers for handling business logic, API calls, and other backend operations, all with minimal effort.
- **Providers:** Quickly create providers to manage and share state across your app, ensuring a smooth and responsive user experience.
- **Views:** Generate views that form the user interface of your app, following best practices for maintainability and scalability.
- **Pages:** Build entire pages for your app with just a few clicks, streamlining the development process and reducing repetitive tasks.
